#### 连接电脑

通过串口连接蓝牙模块（波特率 9600，数据位 8，停止位 1，无校验位）。

发送 AT，会收到 OK。

发送 AT+NAME，会收到 模块的蓝牙名。

#### 连接手机

开启`蓝牙`和`定位`，安卓 APP 中选择透传，UUID 设置为常规模组，然后点搜索BLE（仅发送）或点击搜索SPP（配对码1234，可收发）即可。

连接完成后，模块自动进入透传模式。
